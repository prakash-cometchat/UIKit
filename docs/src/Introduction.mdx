# Introduction

[docz-theme-extended](https://github.com/nejcm/docz-theme-extended) is an extended
default docz gatsby theme.

## Motivation

The default theme is very bare bones. This theme offers some extended
functionality and builds upon the existing UI.

## Configuration

This documantation only describes the extended functionality built on top of the default docz theme.

For more information about extending and configuring the theme please visit the offical
[docz website](https://www.docz.site/).
